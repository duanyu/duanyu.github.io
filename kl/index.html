KL Divergence 数学实例

Forward KL (SFT)

目标：最小化 $\sum P(x) \log \frac{P(x)}{Q(x)}$

P = [0.4, 0.4, 0.2]
Q = [0.5, 0.5, 0.0]
KL = 0.4*log(0.4/0.5) + 0.4*log(0.4/0.5) + 0.2*log(0.2/0.0001) ≈ 0.11

结论：为了不让 $Q(x)=0$ 导致无穷大，Forward KL 会强制 $Q$ 覆盖 $P$ 的所有非零区间，导致分布变宽。

Reverse KL (RLHF)

目标：最小化 $\sum Q(x) \log \frac{Q(x)}{P(x)}$

Q = [1.0, 0.0, 0.0]
P = [0.4, 0.4, 0.2]
KL = 1.0*log(1.0/0.4) + 0 + 0 ≈ 0.91

结论：只要 $P$ 在 $x$ 处有概率，即使 $P(x)$ 很小，$Q$ 也可以为 0。因此 Reverse KL 会大胆抛弃不重要的峰，锁定最高点。
